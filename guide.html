Ponderless Guide

How Ponderless works

Ponderless is a gate-free lap timer for motocross and enduro practice. Just place your iPhone beside the track, ride past the camera, and review the captured passes afterward.

No transponder. No timing gate. Just a phone.

1. Set the line and start detection
2. Ride past the camera
3. Open the pass list after your ride
4. Select passes to calculate lap times

Step-by-step

Step 1

Set the line and start detection

Place your phone where riders pass clearly in front of the camera. Adjust the pass line, start detection, ride past the camera, and then open the pass list after the ride.

Step 2

View passes in time order

The pass list shows captured pass photos sorted by time. Tap each pass to inspect it more closely.

Step 3

Select passes and calculate lap times

Select two or more passes from the same rider to calculate lap times and check the lap result summary.

Pro

Use Pro features for easier matching and session management

Pro adds video mode, rider direction setting, auto match, and session save features for easier workflow.

Pro

Review Auto Match suggestions

Check the suggested candidates, choose the correct passes, and save the result as a session.

Pro

Save and manage sessions

Saved sessions help you keep lap results organized and easier to revisit later.
